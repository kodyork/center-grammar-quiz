センター試験 文法語法クイズ 2016

空欄に入れるのに最も適当なものを一つ選べ。(全10問)

2016年度 (本試験)

問1

The train [ 317 ] when I reached the platform, so I didn't have to wait in the cold.

答え: 1. had already arrived
私がプラットフォームに着いた（過去）よりも前に、電車は到着していた（大過去）ことを示すため、過去完了形 (had done) を使う。

問2

[ 318 ] Tokyo has a relatively small land area, it has a huge population.

答え: 1. Although
Although S+V ... で「〜だけれども」。Despite は前置詞なので名詞が続く。But, However は文頭でこのように使えない。

問3

Children [ 319 ] by bilingual parents may naturally learn two languages.

答え: 2. brought up
Children (who are) brought up by ... の関係代名詞+be動詞が省略された形。子供たちは「育てられる」側なので過去分詞 brought up を使う。

問4

My sister was not a serious high school student, and [ 320 ].

答え: 4. neither was I
前の否定文（was not）を受けて「私もまた〜でなかった」と同意する場合、neither (nor) + 助動詞 + S の語順になる。

問5

Before the movie begins, please [ 321 ] your mobile phone is switched off.

答え: 2. make sure
make sure (that) ... で「…であることを確かめる」。

問6

We have made good progress, so we are already [ 322 ] schedule.

答え: 1. ahead of
ahead of schedule で「予定より進んで」。↔ behind schedule 「予定より遅れて」。

問7

Thanks to their [ 323 ] comments after my presentation, I felt very relieved.

答え: 1. friendly
comments (名詞) を修飾するのは形容詞。friendly は -ly で終わるが「友好的な」という意味の形容詞。

問8

( A ) you've completed this required class, you ( B ) be able to graduate. [ 324 ]

答え: 3
「この必須クラスを完了するまで (Until)、卒業することはできない (won't)」。If (もし) だと「卒業できない」となり文脈に合わない。

問9

Wood ( A ) be used as the main fuel, but nowadays fossil fuels ( B ) widely. [ 325 ]

答え: 1
(A) used to be ... で「かつては〜だった」（過去の習慣・状態）。(B) nowadays (今日では) とあるので現在形。化石燃料は「使われる」側なので受動態 are used。

問10

( A ) so considerate ( B ) him to come and see his grandmother in the hospital every day. [ 326 ]

答え: 4
It is [形容詞] of [人] to do ... で「〜するとは[人]は[形容詞]だ」。considerate (思いやりのある) は人の性質を表す形容詞なので of を使う。
