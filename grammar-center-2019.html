センター試験 文法語法クイズ 2019

空欄に入れるのに最も適当なものを一つ選べ。(全10問)

2019年度 (本試験)

問1

Casey was getting worried because the bus going to the airport was clearly [ 347 ] schedule.

答え: 2. behind
`behind schedule` で「予定より遅れて」。`ahead of schedule`（予定より進んで）の対義語です。

問2

If you are in a hurry, you should call Double Quick Taxi because they usually come in [ 348 ] time.

答え: 3. no
`in no time` で「すぐに、あっという間に」という意味のイディオムです。

問3

After [ 349 ] dropping the expensive glass vase, James decided not to touch any other objects in the store.

答え: 1. almost
`almost` は副詞で「ほとんど〜するところだった」。`almost dropping` で「落としそうになって」。

問4

We should make the changes to the document quickly as we are [ 350 ] out of time.

答え: 2. running
`run out of ...` で「…を使い果たす、…がなくなる」。`running out of time` で「時間がなくなりかけている」。

問5

It was impossible to [ 351 ] everyone's demands about the new projects.

答え: 4. meet
`meet demands` で「要求を満たす、要求に応える」。

問6

Write a list everything you need for the camping trip. [ 352 ], you might forget to buy some things.

答え: 3. Otherwise
`Otherwise` は「さもなければ」。命令文の後で「（もしそうしなければ）忘れるかもしれない」。

問7

Text messaging has become a common [ 353 ] of communication between individuals.

答え: 3. means
`means` は単数形も複数形も `means` で、「手段、方法」という意味の名詞です。`a common means` で「一般的な手段」。

問8

I was ( A ) when I watched the completely ( B ) ending of the movie. [ 354 ]

答え: 2
(A) 私（人）が「衝撃を受けた」ので `shocked`（過去分詞）。 (B) 結末（モノ）が「驚くべき」ものだったので `surprising`（現在分詞）。

問9

( A ) is no ( B ) the increase in traffic on this highway during holidays. [ 355 ]

答え: 4
`There is no ...ing` で「〜することはできない」というイディオムです。`avoiding` が動名詞として使われます。

問10

The police officer asked the witness ( A ) the situation as ( B ) as possible. [ 356 ]

答え: 4
(A) `ask 人 to do` で「人に〜するよう頼む」。 (B) `as ... as possible` の `...` には、動詞 `describe` を修飾する副詞 `accurately`（正確に）が入ります。
